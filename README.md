## About The Project

[![Product Name Screen Shot][product-screenshot]](https://todo-nextjs-5s4h.onrender.com/)

Todo-Nextjs is a task management app designed to help users keep track of their tasks efficiently. It features a clean and intuitive user interface, allowing users to easily add, edit, and manage their tasks. With drag-and-drop functionality, users can effortlessly organize their tasks into different categories. Additionally, the app includes user authentication and data storage with MongoDB, enabling secure access to personalized task lists.


<p align="right">(<a href="#readme-top">back to top</a>)</p>


### [Live Project Link](https://todo-nextjs-5s4h.onrender.com/) 
### Built With


* **Next.js 14:** The latest version of Next.js used for building the application with its App Router and advanced features.

* **Tailwind CSS:** A utility-first CSS framework used for styling the application with a modern and responsive design.

* **@react-beautiful-dnd:** Provides drag-and-drop functionality for a smooth and interactive user experience.

<p align="right">(<a href="#readme-top">back to top</a>)</p>


## Features

* **Modern UI:** A responsive and visually appealing design built with Tailwind CSS for an enhanced user experience.

* **Drag-and-Drop Functionality:** Powered by @react-beautiful-dnd, enabling users to reorder tasks and move them between different categories seamlessly.

* **Task Management:** Users can add new tasks, edit existing ones, and delete tasks as needed.

* **Data Storage:** MongoDB is used to store user data and tasks, ensuring reliable and scalable data management.

* **User Authentication:** Secure login and registration features to allow users to manage their own task lists.

* **Responsive Design:** Optimized for various screen sizes to ensure a smooth experience on both desktop and mobile devices.

* **Categorization:** Tasks are organized into different sections (e.g., Todo, Doing, Done) to help users track their progress effectively.



